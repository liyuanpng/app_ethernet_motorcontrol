SOMANET Motor Control ADC Interface Component
=============================================

:scope: General Use
:description: ADC interface for SOMANET IFM-Drive-DC100/300
:keywords: SOMANET
:boards: SOMANET IFM-Drive-DC100, SOMANET IFM-Drive-DC300


Key Features
------------

  * Support for external trigger

Description
-----------

The component provides an ADC server which drives the ADC of the IFM Drive-DC100/300 boards. The server acquires analog input data in a loop. The component also provides client functions to obtain data from the ADC server.
